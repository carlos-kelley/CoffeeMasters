# Coffee Masters
Built entirely Swift. Order delicious coffee!

## Prerequisites

Before you get started, make sure you have the following software installed on your computer/phone:

- [Xcode] (https://developer.apple.com/xcode/

## Installation

1. Clone the repo: 
   ```sh
   git clone https://github.com/carlos-kelley/CoffeeMasters.git
   ```
2. Open the project file in Xcode.

3. Choose your platform and hit Command-R to build and run in simulator.
